## 📌 How to Explore

```bash
# 1. Clone the repository
git clone https://github.com/your-username/Frontend-Dev.git

# 2. Switch to a branch
git checkout intro-to-web-dev

# 3. Follow branches in order for structured learning
git checkout html
git checkout css
# ... and so on
```

> 💡 **Tip:** Complete each branch before jumping ahead — later topics like Async JS and React build directly on earlier fundamentals like DOM and ES6 features.
